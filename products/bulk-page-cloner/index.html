How It Works

Open Bulk Page Cloner from Apps, the /bulkpages macro, or the ••• menu, then follow these four steps.

1

Select a Page as a Template

Search for any Confluence page or paste the URL of the page you want to clone.

2

Bulk Cloning

Using the slider choose from 1–25 pages then add the titles to the cloned pages.

3

Pick Location

Select a Confluence space, then choose Create Child Page, Create Parent Page, or Create Parent & Child to place your clones.

4

Confirm & Go Live

Review the success page to see every generated link so you can jump straight into your new pages.
