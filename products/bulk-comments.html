Features

Everything you need to comment at scale

Built on Atlassian Forge — no external servers, no data leaving your Jira instance.

One Comment, or One Per Ticket

Post the same rich-text comment — bold, lists, @mentions — to every issue, or write a different note per ticket in one screen. Either way, send to up to 25 tickets at once; blank rows are skipped.

Copy a Comment Exactly, Attachments Included

Paste the link to an existing comment and copy it exactly as-is onto multiple tickets — images, videos and file attachments are copied automatically, with no manual re-uploading.

Jump In From Any Ticket

Copy a comment straight from any issue's ••• actions menu or the Activity panel's "Bulk Comments" tab — the most recent comment is auto-selected for a fast copy.

Secure & Atlassian-Approved

Built on Atlassian Forge — no external servers, no third-party data transfers. All data stays inside Atlassian's infrastructure.

Highlights

Commenting without the copy-paste

The part everyone quietly dreads — pasting the same update into ticket after ticket — collapses into a single send.

Write · Per-ticket override

A different comment on every ticket

Working through a whole project? Write one comment for every ticket, then click any single issue to write it a different comment — formatting, @mentions and all — without leaving the page. Everything posts in one send.

Write · One for all

Write it once, post it twenty-five times

Draft a rich-text comment — bold, lists, @mentions — and send it to as many as 25 tickets in one go. The update that used to mean twenty-five tabs and twenty-five pastes goes out in a single click, worded identically every time.

Copy a comment

Copy a comment, media and all

Point at a comment that already exists and duplicate it exactly — text, formatting, images, video, and file attachments all carry across. Media moves directly between tickets, so nothing is re-uploaded and nothing is quietly left behind.
