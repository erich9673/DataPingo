üöÄ Getting Started

Installation

  1. Navigate to the Atlassian Marketplace
  2. Search for "Bulk Report Creator for Confluence"
  3. Click "Install" and follow the prompts
  4. The app will be available in your Confluence instance within minutes

First Use

Quick Start:
  1. Access the App: Type /bulkreport in any Confluence page
  2. Select from Macro Menu: Choose "Bulk Report Creator" from the dropdown
  3. Start Configuration: The 4-step wizard will guide you through setup

üìñ User Guide

Step-by-Step Workflow

Step 1: Space and Page Selection

  1. Choose Target Space: Select the Confluence space for your reports
  2. Select Parent Location: Choose folder or parent page structure
  3. Validation: System confirms permissions and access rights

Step 2: Template Selection

  1. Browse Categories: Explore 15+ business function categories
  2. Preview Templates: View structure and content organization
  3. Select Multiple: Choose one or more templates for your reports
  4. Customization Preview: See available modification options

Step 3: Metrics and Configuration

  1. KPI Selection: Choose relevant key performance indicators
  2. OKR Alignment: Connect objectives and key results
  3. Custom Metrics: Add organization-specific measurements
  4. Time Estimates: Set realistic completion timeframes
  5. Report Naming: Define consistent naming conventions

Step 4: Generation and Scheduling

  1. Review Configuration: Confirm all settings and selections
  2. Schedule Options: Set immediate or future generation
  3. Bulk Generation Rules: Configure multiple report creation
  4. Final Execution: Generate reports with monitoring dashboard

üö® Troubleshooting

Common Issues

Installation Problems

Issue: App not appearing after installation

Solution: Refresh browser cache and re-login to Confluence

Alternative: Check Confluence app management for installation status

Configuration Issues

Issue: Templates not loading

Solution: Check internet connection and refresh the page

Alternative: Clear browser cache and try again

Generation Problems

Issue: Reports not creating

Solution: Check Confluence storage limits and permissions

Alternative: Try generating fewer reports at once
