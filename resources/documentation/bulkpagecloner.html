Bulk Page Cloner for Confluence

Duplicate up to 25 Confluence pages in one guided flow — with a live clone summary and full light & dark mode.

Overview

Bulk Page Cloner is an Atlassian Forge app for Confluence Cloud that duplicates up to 25 pages from a single template in one guided flow. Each clone is an exact copy — content, formatting, images, video, macros, labels, and attachments included — ready to edit independently. A live clone summary builds as you go, and the whole app supports full light and dark mode matched to your Confluence theme.

Pro Tip: This tool is perfect for teams that need to maintain consistency across multiple Confluence spaces or quickly replicate content structures for new projects.

Full Guide

Open the App (3 ways)

  1. Click Bulk Page Cloner for Confluence under Apps on the left side of your screen.
  2. Type /bulkpage in any Confluence page and select Bulk Page Cloner for Confluence.
  3. Open the ••• menu on any page and select Create Bulk Pages.

The app walks you through a four-step flow — Source, Copies, Location, Done — with a progress bar at the top. From the Copies step onward, a live clone summary shows your source, copy count, and destination, updating as you go so you can review everything before anything is created.

Step 1 — Source

Choose the page you want to clone.

  • Search page titles to find any page in your Confluence instance, then click Select.
  • Use the Filter by space dropdown to narrow the list.
  • Have a link? Use Paste a page URL to jump straight to a specific page.

Step 2 — Copies

Decide how many copies to make and what each one is called. Every copy is an exact duplicate of the source — content, formatting, images, video, macros, labels, and attachments included.

  1. Set the Number of copies (1–25) with the slider or the input box.
  2. Name each copy in the title fields.

Shortcuts

  • Auto-number — Name one page (e.g. "Report 1") and the rest fill in sequentially. Recognizes weekly and monthly date patterns too.
  • Copy first to all — Duplicate the first title across every field.
  • Clear all — Empty all title fields at once.

Heads-up before you create: the app warns you when a source page @mentions people (each copy notifies them again), for large batches of 10+ pages, and for duplicate titles before Confluence auto-numbers them.

Step 3 — Location

Choose where the copies are created.

  1. Pick the destination Space.
  2. Choose how they're organized:
    • As a child of an existing page — Copies become child pages under a parent you pick.
    • As parent pages in [Space] — Copies have no child; they sit at the space root.
    • As child pages of a new parent — Create a new parent page, then add the copies beneath it.

The clone summary now shows the full destination path. When it looks right, click Create pages.

Step 4 — Done

A confirmation screen lists every created page with a direct Open link, so you can jump straight in to start editing — or clone more pages in one click.

Latest Updates in v7.1.0

Version 7.1.0 (released July 6, 2026) is a ground-up redesign:

  • Guided 4-step flow — A clear Source → Copies → Location → Done flow with a progress bar and a live clone summary that builds as you go, so you review everything before you create.
  • Full dark mode — Light and dark theme support that matches your Confluence theme and switches live, no reload needed.
  • Smarter & safer — A heads-up when a source page @mentions people (each copy notifies them again), plus warnings for large batches (10+ pages) and duplicate titles before Confluence auto-numbers them.
  • Search-first picking — A searchable space and parent-page selector that loads every page in the space, an editable copy count, and clearer wording using Confluence's own parent/child terms.
  • Faster under the hood — Caching and rendering improvements across the cloning pipeline.

System Requirements

  • Active Confluence Cloud instance
  • Modern web browser (Chrome, Firefox, Safari, Edge)
  • Active internet connection
  • Appropriate Confluence permissions

Permissions Required

  • Read and write Confluence pages (to load templates and create pages)
  • Space administration rights (for space-level operations)

Cloud only: Bulk Page Cloner is built on Atlassian Forge and runs exclusively on Confluence Cloud. Confluence Server and Data Center are not supported.
