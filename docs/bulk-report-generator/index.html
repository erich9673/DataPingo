Installation Guide

Install from Atlassian Marketplace

  1. Access Atlassian Marketplace

    Visit the Bulk Report Generator page on Atlassian Marketplace.

  2. Install the App

    Click "Get it now" and follow the installation prompts for your Confluence instance.

  3. Grant Permissions

    The app requires permissions to create and edit pages in your Confluence spaces.

  4. Access Admin Dashboard

    After installation, navigate to your Confluence administration to access the Bulk Report Generator admin dashboard.

    Admin Dashboard: View previously created reports (click 'Open' to access them) or start creating new reports

  5. Verify Installation

    Go to any Confluence page and type /bulkreport to verify the macro is available.

Note: This app is built on Atlassian Forge platform, ensuring secure processing within your Confluence environment with no external data transmission.

Troubleshooting

Common Issues

Macro not appearing when typing /bulkreport

Solution:

  • Verify the app is installed in your Confluence instance
  • Check if you have permissions to use macros
  • Try refreshing the page and typing the command again

Cannot create reports in selected space

Solution:

  • Ensure you have "Create" permissions in the target space
  • Contact your Confluence administrator for space permissions
  • Try selecting a different space where you have full access

Template editing not working

Solution:

  • This feature is only available for Simple Templates
  • Upload Templates skip the editing step
  • Clear browser cache and try again

Scheduled reports not generating

Solution:

  • Verify the macro is still present on the original page
  • Check space permissions haven't changed
  • Contact support if issues persist
