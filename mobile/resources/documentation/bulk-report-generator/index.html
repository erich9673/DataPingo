‚ùì FAQ

Is my data secure with this app? ‚ñº

Yes, absolutely. Built on Atlassian Forge platform, all processing happens within your Confluence environment with no external data transmission.

Can I use this with Confluence Server/Data Center? ‚ñº

Currently designed for Confluence Cloud only. Server and Data Center versions are not supported at this time.

How many reports can I generate at once? ‚ñº

No strict limit, but we recommend batches of 50-100 reports for optimal performance. For larger volumes, use smaller batches or scheduled generation.

Can I modify templates after creating reports? ‚ñº

Yes, you can modify templates anytime. Changes only apply to new reports generated after modification.
